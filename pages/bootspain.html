Bootstrap es una herramienta muy útil para desarrollar páginas web responsive, lo que permite que los sitios se vean bien y funcionen correctamente en diferentes dispositivos y pantallas. Además, ofrece una amplia variedad de herramientas y componentes para que los desarrolladores puedan crear sitios web atractivos y funcionales con mayor rapidez, facilidad y eficiencia.
Interactividad
Bootstrap proporciona una gran interactividad en la página a través de una serie de componentes que facilitan la interacción con el usuario, como menús de navegación, barras de progreso y controles de página.
Diseño adaptable
El objetivo principal de Bootstrap es construir sitios web responsive para dispositivos móviles. De una manera muy simple y organizada, permite diseñar páginas especialmente adaptadas para funcionar en smartphones, tablets y ordenadores.
Biblioteca de componentes
Bootstrap ofrece una gran cantidad de componentes para mejorar la comunicación con el usuario, como diferentes tipos de alertas, un carrusel deslizante para visualizar imágenes fácilmente y una barra de navegación que permite múltiples configuraciones.
Ventajas
  • Fácil de usar y ahorra tiempo - Permite diseñar un sitio web usando diferentes combinaciones de HTML, CSS y JavaScript. Utiliza tecnologías modernas como HTML5, CSS3 y jQuery, e integra con bibliotecas de JavaScript. Viene con estilos y plantillas predefinidas. Viene con documentación completa que hace que sea fácil de usar.
  • Responsive - Sus plantillas son fácilmente personalizables y adaptables. Incluye un útil sistema de cuadrícula para la disposición en columnas que se adapta a diferentes tamaños de pantalla y dispositivos.
Desventajas
  • Compatibilidad - Cambiar de versión puede ser difícil si se han realizado modificaciones profundas. Se recomienda trabajar con Bootstrap desde el inicio del proyecto para evitar problemas de compatibilidad. No es un marco de trabajo ligero y algunas funcionalidades requieren el uso de JavaScript y jQuery.
  • Restricciones de diseño - Es necesario adaptar el diseño a un sistema de cuadrícula de 12 columnas, lo que puede requerir cambios específicos en los estilos predeterminados de Bootstrap.
  • Puede ser pesado - Los archivos generados por Bootstrap y los resultados finales pueden ser bastante pesados y provocar tiempos de carga más lentos, así como problemas de consumo de batería.
  • Diseño poco original - Sin muchas modificaciones, todo lo construido con Bootstrap tendrá un aspecto muy similar.
Bootstrap se puede instalar fácilmente visitando la página oficial y descargando los archivos del paquete. Los archivos de instalación incluyen hojas de estilo CSS, archivos JavaScript y fuentes, y pueden adaptarse a cualquier tipo de proyecto web. Para trabajar con Bootstrap, simplemente agregue las hojas de estilo y el archivo JavaScript en el encabezado o antes de la etiqueta de cierre del archivo HTML. Como alternativa, se puede usar utilizando enlaces de Bootstrap CDN que se pueden incluir en el proyecto de forma gratuita.
Los sitios construidos con Bootstrap se ajustarán automáticamente al tamaño de la pantalla en la que se vean. Esto es posible gracias a la inclusión de las características de diseño responsive en el Sistema de rejilla de Bootstrap.
El sistema de rejilla divide la pantalla en doce columnas virtuales y permite a los desarrolladores especificar cómo mostrar los elementos en cada tamaño de pantalla. Por ejemplo, en una pantalla pequeña, ciertos elementos pueden ser especificados para apilarse verticalmente para facilitar su lectura y navegación en dispositivos móviles.
Para implementar un diseño responsive en Bootstrap, se pueden usar las clases de tamaño de columna, como: col-sm para pantallas pequeñas col-md para pantallas medianas, y col-lg para pantallas grandes. Estas clases se pueden aplicar a cualquier elemento HTML, como un div o una imagen, para definir cómo se mostrará en diferentes tamaños de pantalla..
Por ejemplo, se puede especificar que un elemento tenga un ancho de 12 columnas (ancho completo de la pantalla) en dispositivos móviles, pero solo 3 o 4 columnas en pantalla.
Bootstrap ofrece una gran cantidad de componentes preconstruidos que los desarrolladores web pueden utilizar en sus proyectos. Estos componentes incluyen menús de navegación, carruseles, tablas y muchos otros. En su página oficial, Bootstrap ofrece ejemplos de estos componentes, lo que facilita su implementación. Algunos de los componentes más utilizados son:
  • carousel
  • navbar y navs
  • jumbotron
  • breadcrumbs
  • pagination
  • tooltips
  • forms
  • cards
Por ejemplo, un navbar y los componentes nav permiten crear una barra de navegación responsive con diferentes elementos como enlaces y desplegables. Los elementos de la barra de navegación se crean utilizando listas HTML y se estilizan con clases específicas de Bootstrap. El JavaScript incorporado de Bootstrap se puede utilizar para activar la funcionalidad de desplegable en la barra de navegación.
Hay dos diseños principales para Bootstrap:
Diseño Fijo
Este utiliza la clase .container de Bootstrap para el diseño. El diseño fijo tiene valores de ancho específicos en píxeles que cambian con la ayuda de consultas de medios, lo que hace que el sitio sea receptivo. El diseño fijo cambia de tamaño en intervalos siguiendo los valores de píxeles especificados.
Diseño Fluido
Este utiliza la clase .container-fluid de Bootstrap para el diseño. Este diseño utiliza valores proporcionales como porcentajes para bloques de contenido, imágenes o cualquier otro elemento. El diseño fluido cambia de tamaño continuamente a medida que cambias el ancho de tu navegador por cualquier cantidad, sin dejar espacio vacío adicional en los lados. Por lo tanto, se llama "diseño fluido".
Los contenedores son el elemento de diseño más básico en Bootstrap y son necesarios al utilizar nuestro sistema de cuadrícula predeterminado. Bootstrap viene con tres contenedores diferentes.
Contenedores de diseño fijo:
  • .container, que establece un ancho máximo en cada punto de interrupción receptivo
  • .container-{punto de interrupción}, que es ancho: 100% hasta el punto de interrupción especificado. El punto de interrupción puede ser (sm/md/lg/xl). Ver aquí para más detalles.
Contenedor de diseño fluido:
  • .container-fluid, que tiene un ancho: 100% en todos los puntos de interrupción
Bootstrap incluye estilos de botón predefinidos, cada uno con su propio estilo y/o propósito semántico:
  • Proporciona un mayor peso visual a un botón mientras representa una acción primaria que los visitantes deben realizar en la página. Ejemplo: página de inicio.
  • Proporciona un mayor peso visual a un botón mientras representa una acción secundaria que los visitantes deben realizar en la página. Ejemplo: demo.
  • Indica una acción exitosa o positiva. Ejemplo: al completar un formulario.
  • Una acción potencialmente peligrosa. Ejemplo: para asegurarse de que los visitantes revisen su información antes de enviarla.
  • Representa una advertencia. Por ejemplo, el botón de advertencia puede indicar que algunos campos de formulario requeridos fueron omitidos.
  • Una alerta informativa. Por ejemplo, en un formulario de registro de cuenta, un botón de información podría indicar que un usuario puede hacer clic para editar su perfil.
  • Una gran opción para páginas web con colores de fondo oscuros.
  • Funciona bien para páginas web con colores de fondo claros.
